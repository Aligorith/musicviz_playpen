Response
========

Those regressions happened because the last version accidentally replaced the earlier dense spectrogram renderer with a simplified placeholder renderer (`cachedSpectrogram.push({freq...})`) that only plots sparse dots instead of FFT magnitude bins.

Here are the fixes to restore the previous good rendering while keeping the newer playback/UI features.

---

# 1. Restore dense spectrogram rendering

Replace the ENTIRE `redrawSpectrogram()` with this:

```js id="restore_dense_spectrogram"
function redrawSpectrogram(){

    specCtx.clearRect(
        0,
        0,
        canvas.width,
        canvas.height
    );

    if(!cachedSpectrogram.length)
        return;

    const xScale =
        canvas.width /
        cachedSpectrogram.length;

    for(
        let i=0;
        i<cachedSpectrogram.length;
        i++
    ){

        const spec =
            cachedSpectrogram[i];

        for(
            let b=1;
            b<spec.length;
            b++
        ){

            const freq =
                b *
                (
                    cachedSampleRate /
                    cachedFFTSize
                );

            if(
                freq < MIN_FREQ ||
                freq > MAX_FREQ
            ) continue;

            const y =
                freqToY(freq);

            const mag =
                spec[b];

            const alpha =
                Math.min(
                    1,
                    (
                        Math.log10(
                            1 + mag
                        ) / 2.1
                    ) *
                    brightnessBoost
                );

            const c =
                colorForFreq(freq);

            specCtx.fillStyle =
                `rgba(
                    ${c[0]},
                    ${c[1]},
                    ${c[2]},
                    ${alpha}
                )`;

            specCtx.fillRect(
                i * xScale,
                y,
                Math.ceil(xScale)+1,
                1
            );
        }
    }
}
```

# 3. Restore C markers

The previous draw loop was lost.

Add this ENTIRE function back ABOVE `render()`:

```js id="restore_c_markers"
function drawCMarkers(){

    for(let o=0;o<9;o++){

        const midi =
            12 + o*12;

        const freq =
            aTuningHz *
            Math.pow(
                2,
                (midi - 69)/12
            );

        if(
            freq < MIN_FREQ ||
            freq > MAX_FREQ
        ) continue;

        const y =
            freqToY(freq);

        ctx.strokeStyle =
            "rgba(255,255,255,0.28)";

        ctx.beginPath();

        ctx.moveTo(0,y);
        ctx.lineTo(16,y);

        ctx.stroke();

        ctx.fillStyle =
            "rgba(255,255,255,0.75)";

        ctx.font =
            o === 4
            ? "bold 12px monospace"
            : "11px monospace";

        ctx.fillText(
            `c${o}`,
            20,
            y + 4
        );
    }
}
```

Then INSIDE `render()` AFTER:

```js id="timeline_call"
drawTimeline();
```

add:

```js id="c_marker_call"
drawCMarkers();
```
