musicviz_playground
===================

Git repo for hosting my various experimental attempts to build a novel music visualisation engine / tool
that behaves in the ways that I want.


Original Author:
Dr Joshua Leung  (@Aligorith)


Contents
========

* **01_pitch_color_map** - First steps to define what the mapping between pitches to colours used to represent them

* **02_ffmpeg_showcqt** - The set of Windows .BAT scripts to run FFMPEG's "showcqt" to produce some promising
  "spectrogram" like scrolling visualisations.
